# 🏠 California Housing Price Prediction - Task 2

## 📌 Project Overview

This project is part of the **Artificial Intelligence & Machine Learning Internship - Task 2**.

The objective of this task is to compare multiple regression models on the California Housing Dataset by applying feature scaling and evaluating model performance using RMSE and R² Score.

---

## 🎯 Objective

- Load the California Housing Dataset
- Perform data preprocessing
- Apply Feature Scaling using StandardScaler
- Train multiple regression models
- Compare model performance
- Select the best-performing model
- Visualize predictions

---

## 📂 Dataset

**Dataset:** California Housing Dataset

The dataset is available through the Scikit-Learn library and contains housing-related information such as:

- Median Income
- House Age
- Average Rooms
- Average Bedrooms
- Population
- Average Occupancy
- Latitude
- Longitude

**Target Variable**

- Median House Value

---

## 📚 Machine Learning Models

The following regression models were implemented:

- Linear Regression
- Ridge Regression
- Decision Tree Regressor

---

## ⚙️ Workflow

1. Import required libraries
2. Load California Housing Dataset
3. Convert dataset into a DataFrame
4. Split dataset into training and testing sets
5. Apply Feature Scaling
6. Train Linear Regression
7. Train Ridge Regression
8. Train Decision Tree Regressor
9. Evaluate models using RMSE and R² Score
10. Compare model performance
11. Visualize Actual vs Predicted values
12. Select the best-performing model

---

## 📊 Model Evaluation

The models were evaluated using:

- Root Mean Squared Error (RMSE)
- R² Score

The comparison helps determine which model predicts house prices most accurately.

---

## 📈 Visualizations

This project includes:

- Actual vs Predicted Scatter Plot
- Model Comparison Bar Chart

---

## 🏆 Result

Among the three models tested:

- Decision Tree Regressor achieved the **highest R² Score**
- Decision Tree Regressor achieved the **lowest RMSE**
- Decision Tree Regressor was selected as the **best-performing model**
